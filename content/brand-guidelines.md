# SESAM — Brand Guidelines (for the landing page)

> Quick-reference rules for keeping the landing page visually and tonally consistent
> with @sesam.sa.

## 1. Voice
- **Primary language:** Arabic (Saudi / Gulf).
- **Secondary:** English — only for brand name, short evocative phrases, hashtags.
- **Tone:** warm, slow, refined. Never loud or fast-food-style.
- **Punctuation in copy:** use light emoji (🥐✨💛🎀) the way the Instagram captions do — sparingly.

## 2. Logo
- **Type:** the SESAM avatar is the de-facto logo mark (no separate wordmark file).
- **File used:** `images/profile/avatar.jpg` (150×150 JPEG from the account).
- **Placement:** top-left of header, circular, gold hairline ring on cream background.
- **Wordmark:** use the text `SESAM` in display serif next to the mark when useful.

## 3. Colors (do not deviate)

| Token | Hex | Where to use |
|---|---|---|
| `primary` (olive) | `#77715b` | Headings, primary buttons, dividers |
| `primary-dark` (espresso) | `#2e2b24` | Body text, footer bg |
| `secondary` (sand) | `#c5bea4` | Section dividers, secondary buttons |
| `accent` (gold) | `#b89968` | Hover, underlines, badges |
| `bg` (cream) | `#fbf6f0` | Page background |
| `surface` | `#ffffff` | Cards |

## 4. Typography

- **Primary (Arabic + Latin):** **IBM Plex Sans Arabic** — used for ALL Arabic text
  and the entire UI (headings, body, nav, buttons, badges). Modern, geometric,
  excellent Arabic glyph set with proper joining. Weights used: 300 / 400 / 500 / 600 / 700.
- **Body (Latin fallback):** Jost — used for Latin paragraphs, small caps UI labels,
  navigation EN labels.
- **Serif accent (Latin only):** Cormorant Garamond — used sparingly for italic
  English hero subtitle, brand name EN, badge numbers, and italic emphasis.
- **Sizes (desktop):** H1 56–72 px, H2 36–44 px, H3 24–28 px, body 16–18 px.
- **Sizes (mobile):** scale down ~30–40%.

## 5. Imagery
- All images must come from **/images/posts/** and **/videos/posts/** — both are
  100% from @sesam.sa Instagram posts.
- **Post-processing rule:** none. Do not add heavy filters. Keep the original warm
  cream / olive mood.
- **Border-radius:** 12–18 px on cards; full circle on avatars.
- **Aspect ratio:** prefer 4:5 or 1:1; never stretch.

## 6. Layout
- **Direction:** RTL (Arabic first).
- **Reading flow:** right → left.
- **Grid:** 12-col, max-width 1200 px, generous gutters.
- **Whitespace:** generous. Let the visuals breathe.

## 7. Motion
- **No autoplay video.**
- Subtle fade-in on scroll.
- Hover scale `1.03` max on cards.
- Reels open via Instagram embed (no autoplay).

## 8. Accessibility
- Alt text on every image (Arabic + English captions).
- `prefers-reduced-motion` disables all transitions.
- Color contrast: `primary-dark` on cream passes WCAG AA.

## 9. Forbidden
- Stock photos.
- Copy that "sells" too hard.
- Bright primary colors (red / blue / green) outside the brand palette.
- Embedded reels auto-playing with sound.
